# Node.js-Expess-MongoDB-Shopping side Employee CRUD with EJS

Content discussed : 
 - Node express with EJS for fromtend framwork 
 - User loggin / logout, user list feature
 - Insert, Update and Delete for user etc with Node and Mondodb

## How to setup

```
$ git clone git@github.com:codexhunter4u/employe-crud-nodejs-express.git
$ cd node-restfull-api
$ npm install
$ edit config.env
```

 ## How to use ?
 
```
$ cd employe-crud-nodejs-express
$ nodemon app.js or node app.js
```
